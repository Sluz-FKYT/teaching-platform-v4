## 3. 使用建议

- 截图页适合放在“界面展示”和“流程走查”章节
- 若后续你补入更准确的新截图，可直接替换本说明中的占位资源
- 结构说明优先参考 `design/html/`，视觉占位优先参考 `design/screenshots/`

## 4. 不建议直接当成最终事实图的内容

- 设计 HTML 页面
- Stitch 原型过渡截图
- 尚未和真实实现完全一致的中间版本原型

它们适合作为 PPT 中的“页面设计参考”或“界面迭代占位”，但不宜被表述为“最终实现截图”。
